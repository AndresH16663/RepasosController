Ejercicio 2: Clase Coche

Ejercicio 3: Clase Libro

Ejercicio 4: Clase Estudiante

Ejercicio 5: Clase CuentaBancaria

Ejercicio 6: Clase Producto

Ejercicio 7: Clase Mascota
